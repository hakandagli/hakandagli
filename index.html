Hakkımda

Yeni nesil bilgisayar mühendisi

29 Ocak 1999'da Ankara'nın Altındağ ilçesinde doğdum.İlköğretim dönemimi Ankara/Kahramankazan'da tamamlamamın ardından Kazan Mustafa Hakan Güvençer Fen Lisesini kazanarak lise hayatıma başladım.
Lisede bulunduğum sayısal bölümünde derslerin yanısıra başlangıç seviyesi C ve Java programlama ardından web programlamaya başladım.
Lise dönemi ardından Süleyman Demirel Üniversitesi Bilgisayar Mühendisliği bölümüne girmeye hak kazandım. 2021 yılında mezun oldum. Şu an Ankara'da ikamet etmekteyim.
Şuan Html, Css, Javascript, Typescript, Angular, JQuery, Bootstrap, PHP yazılım teknolojilerini kullanıyorum.
Ek iş olarak boş vakitlerimde kişisel ve kurumsal web siteleri yapıyorum.
